Blast Connect - 2

Uttam Blastech

Blast Connect - 2

The BlastConnect is a new system which connects with the Seismograph device for automated collection of all the field data which is monitored by the Seismograph for accurate, undisputed and undisrupted results. The BlastConnect system collects all the monitored and saved data from Seismograph and send all the data to the dedicated computer by using internet protocol. The BlastConnect system having LED indications which allows the operator to understand the status of the system.

Air Quality Monitoring System at Nagpur Mines

Manganese Ore (India) Limited (MOIL)

Air Quality Monitoring System at Nagpur Mines

The system consists of laser particulate matter sensors (PM2.5. PM10), Pressure sensor, Temperature sensor, Relative Humidity sensor. The sensors collects the data and send it to a monitoring system attached with it. The system stores the data with GSP Coordinates of the test points and transmits it in a remote web server through a GSM modem.

Water Quality Monitoring System at Nagpur Mines

Manganese Ore (India) Limited (MOIL)

Water Quality Monitoring System at Nagpur Mines

The system consists of online pH sensor with transmitter, online TDS sensor transmitter in a Metal Sheet panel with inlet and outlet connections. The sensors collects the data and send it to a monitoring system attached with it. The system stores the data with GSP Coordinates of the test points and transmits it in a remote web server through a GSM modem.
